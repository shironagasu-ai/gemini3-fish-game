fish game demo. create with Google Gemini 3

# Deep Sea Predator Simulation 🐟🦈

HTML5 CanvasとJavaScriptで構築された、深海を舞台にしたインタラクティブな生態系シミュレーションです。
Boids（ボイド）アルゴリズムを用いた小魚の群れと、それを追う巨大な捕食者（ボス）の挙動を再現しています。

## ✨ 特徴

### 1. 群体シミュレーション (Boids Algorithm)
小魚たちは「分離（Separation）」「整列（Alignment）」「結合（Cohesion）」の3つのルールに基づき、自律的に群れを形成して泳ぎます。
* **結束の強化**: バラバラにならず、美しいストリーム状の群れを作ります。
* **回避行動**: 捕食者やプレイヤーの干渉から自然に逃げ惑います。

### 2. インタラクティブな操作
プレイヤーは「神の手」としてマウス（またはタッチ操作）で生態系に介入します。
* カーソルを近づけると小魚は蜘蛛の子を散らすように逃げます。
* これを利用して、小魚をボスの進行方向へ「誘導（ヘディング）」し、捕食をサポートするゲーム性があります。

### 3. 捕食者の成長システム
* **追跡ロジック**: ボスは最も近い獲物をターゲットにし、滑らかな旋回挙動で追い詰めます。
* **成長と老化**: 小魚を捕食するたびにボスは巨大化し、その巨体ゆえに移動速度が徐々に低下していきます。
* **Juice（演出）**: 捕食時のパーティクルエフェクト、食べた瞬間の弾力アニメーションなど、触り心地の良さを追求しています。

### 4. 没入感のあるビジュアル
* **深海の表現**: グラデーション背景と、揺らめきながら上昇する気泡（Bubbles）のアニメーション。
* **残像効果**: 描画更新時にうっすらと軌跡を残すことで、スピード感と深海の粘性を表現しています。

## 🚀 実行方法

特別な環境構築は不要です。ブラウザがあればすぐに動作します。

1. このリポジトリをクローンまたはダウンロードします。
2. `fish-game.html` (または保存したファイル名) をモダンブラウザ（Chrome, Firefox, Safari, Edgeなど）で開いてください。

## 🕹️ 操作方法

* **マウス移動 / タッチドラッグ**: 小魚を驚かせて移動させます。群れをボスの口元へ誘導してみましょう。
* **リスタート**: 全ての小魚が食べ尽くされると「ALL FISH EATEN」と表示され、RESTARTボタンで最初からやり直せます。

## 🛠️ 技術スタック

* **HTML5 Canvas**: 描画エンジン
* **Vanilla JavaScript (ES6+)**: ロジック（ライブラリ不使用）
    * Classベースのオブジェクト指向設計
    * Vector（ベクトル）演算による物理挙動
    * `requestAnimationFrame` による滑らかなアニメーションループ
